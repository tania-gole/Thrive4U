# Thrive4U — Website
A professional website for Thrive4U Coaching & Consulting, built for life coach and consultant Sanah Singh Tomar.

## Overview
A clean, mobile-responsive 4-page website designed to establish a professional online presence, showcase Sanah's credentials and coaching philosophy, and give clients a clear path to connect and engage.

## Pages
- **Home** — hero, intro video, call to action
- **About** — bio, coaching philosophy, certifications and credentials
- **My Process** — coaching approach and methodology
- **Testimonials** — client stories

## Tech Stack

| Tool | Purpose |
|---|---|
| Figma | UI design and prototyping |
| Webflow | Site build, hosting, SSL |
| Google Analytics 4 | Traffic and visitor analytics |
| GoDaddy | Domain registration |
| YouTube embed | Homepage intro video |

## Process
1. Discovery call + content collection
2. Figma design — 4 pages, mobile and desktop
3. Webflow build
4. Content population + QA
5. Domain connection + launch
6. Client handover with Loom walkthrough
